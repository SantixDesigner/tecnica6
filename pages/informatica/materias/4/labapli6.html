Laboratorio de Aplicaciones 6°

Docentes

Indicadores de promoción

El alumno deberá

  1. Conocer la historia de la evolución de Internet y la WWW.
  2. Identificar la estructura de un documento HTML.
  3. Lograr la maquetación básica de una página web.
  4. Conocer en profundidad la sintaxis de las etiquetas más utilizadas en el Lenguaje HTML.
  5. Analizar la estructura de páginas web diseñadas, desde el punto de vista del código fuente.
  6. Adquirir conocimientos sobre el lenguaje CSS.
  7. Conectar un documento HTML con una hoja de restillos CSS.
  8. Programar un script con JAVASCRIPT.
  9. Conocer aspectos y expresiones básicas del lenguaje PHP.
  10. Utilizar el lenguaje PHP para conectar la web a un Servidor.
  11. Conocer aspectos generales del lenguaje SQL.

Módulo Temático 1: CONCEPTOS INTRODUCTORIOS SOBRE INTERNET. LA WWW.

Redes de datos. Fundamentos de internet, a nivel hardware y software. Evolución de Internet. Estándar W3C.

Módulo Temático 2: INTRODUCCIÓN AL HTML

Redes de datos. Fundamentos de internet, a nivel hardware y software. Evolución de Internet. Estándar W3C.

Módulo Temático 3: ESTRUCTURACIÓN DE CONTENIDO

Generación de Listas. Mapeado de imágenes. Estructuración y atributos de tablas. Formularios, declaración y manipulación. La etiqueta DIV como contenedor. Posicionamiento. Fuentes en HTML. Vínculos internos. Vínculos externos a videos, imágenes, sitios web o archivos.

Módulo Temático 4: HOJAS DE ESTILO EN CASCADA

Conexión de una hoja de estilo con el código HTML. Selectores. Análisis de interpretación de los navegadores. Declaraciones múltiples. Agrupación de estilos. Herencia.

Módulo Temático 5: DISEÑO DE ELEMENTOS DE LA WEB

Diseño de logos. Banners interactivos. Botoneras. Incrustaciones de objetos web.

Módulo Temático 6: SERVIDORES

Métodos para subir el sitio al servidor (clientes FTP) Introducción a los lenguajes de clientes. Definición de scripts. Programación de scripts. Lenguajes de Servidor. Lenguaje PHP. Expresiones básicas PHP (sintaxis, variables, echo, print, operadores). Bases de datos en SQL. (sintáxis, select, where, crear tabla, crear base de datos)

TRABAJO PRÁCTICO INTEGRADOR ANUAL

El T.P Integrador constará de:

  • Una serie de preguntas teóricas acerca del lenguaje HTML, CSS, PHP y SQL.
  • Resolución de problemas a partir de un código dado.
  • Maquetación de una web utilizando todos los saberes aprendidos.
